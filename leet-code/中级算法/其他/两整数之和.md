# [两整数之和](https://leetcode-cn.com/leetbook/read/top-interview-questions-medium/xwaiag/)

### algorithm:
- 将加法分解为位运算，xor求当前位，and求进位

### code:
```javascript
/**
 * @param {number} a
 * @param {number} b
 * @return {number}
 */
var getSum = function(a, b) {
    if (a === 0) {
        return b
    } else if (b === 0) {
        return a
    } else {
        return getSum(a ^ b, (a & b) << 1)
    }
}
```
