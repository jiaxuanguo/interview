# [最长回文子串](https://leetcode-cn.com/leetbook/read/top-interview-questions-medium/xvn3ke/)

### algorithm:
1. 暴力法，从长到短依次判断每个字串是否回文；时间复杂度O(n<sup>3</sup>)
2. 动态规划：
   - p[i][j]表示从i到j的字串是否回文
   - `p[i][j] = p[i+1]p[j-1] && s[i] === s[j]`
   - `p[i][i] = true`
   - `p[i][i+1] = s[i] === s[i+1]`
   - 时间复杂度O(n<sup>2</sup>)
