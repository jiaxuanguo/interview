# [不同路径](https://leetcode-cn.com/leetbook/read/top-interview-questions-medium/xvjigd/)

### algorithm:
动态规划:
1. p[i,j]表示到达i,j有几条路径
2. `p[i,0] = 1`, `p[0,j] = 1`
3. `p[i,j] = p[i,j-1] + p[i-1,j]`

### code:
```javascript
/**
 * @param {number} m
 * @param {number} n
 * @return {number}
 */
var uniquePaths = function(m, n) {
    const p = []
    for (let i = 0; i < m; i++) {
        p.push([])
        for (let j = 0; j < n; j++) {
            if (i == 0 || j === 0) {
                p[i][j] = 1
            } else {
                p[i][j] = p[i][j - 1] + p[i - 1][j]
            }
        }
    }
    return p[m-1][n-1]
}
```
