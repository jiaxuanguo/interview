# [阶乘后的零](https://leetcode-cn.com/leetbook/read/top-interview-questions-medium/xwehi5/)

### algorithm:
- 2 * 5 = 10, 5 > 2, 结尾的0的数量等于乘数中质因数5的总和
- 计算5的数量时可以遍历每个乘数，也可以不断除以5

### code:
```javascript
/**
 * @param {number} n
 * @return {number}
 */
var trailingZeroes = function(n) {
    let sum = 0
    while (n > 0) {
        n = Math.floor(n / 5)
        sum += n
    }
    return sum
}
```
