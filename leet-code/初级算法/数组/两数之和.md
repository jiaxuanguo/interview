# [两数之和](https://leetcode-cn.com/leetbook/read/top-interview-questions-easy/x2jrse/)

### algorithm:
1. 暴力遍历所有两两组合，直到找到符合要求的两个数；空间复杂度O(1)，时间复杂度O(n<sup>2</sup>)
2. 排序数组，暴力遍历中的第二次遍历可用二分查找，但需记录数在原始数组中的位置；空间复杂度O(n)，时间复杂度O(nlogn)
3. 用HashMap记录遍历过的数和位置，可以避免暴力遍历中的第二次遍历；空间复杂度O(n)，时间复杂度O(n)

### code:
```javascript
/**
 * @param {number[]} nums
 * @param {number} target
 * @return {number[]}
 */
var twoSum = function(nums, target) {
    const map = new Map()
    for(let i = 0; i < nums.length; i++) {
        const num = nums[i]
        const otherNum = target - num
        const otherIndex = map.get(otherNum)
        if (otherIndex != null) {
            return [i, otherIndex]
        } else {
            map.set(num, i)
        }
    }
    throw new Error('cannot find pair')
};
```
