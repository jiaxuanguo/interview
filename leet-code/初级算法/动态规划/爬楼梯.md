# [爬楼梯](https://leetcode-cn.com/leetbook/read/top-interview-questions-easy/xn854d/)

### algorithm:
- p[0] = 1
- p[1] = 1
- p[i] = p[i-1] + p[i-2]

### code:
```javascript
/**
 * @param {number} n
 * @return {number}
 */
var climbStairs = function(n) {
    let p = [1, 1]
    for (let i = 2; i <= n; i++) {
        p[i] = p[i - 1] + p[i - 2]
    }
    return p[n]
}
```
