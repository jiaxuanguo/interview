# [3的幂](https://leetcode-cn.com/leetbook/read/top-interview-questions-easy/xnsdi2/)

### algorithm:
递归：
  1. 小于1的整数都不是3的幂
  2. 1是3的0次幂
  3. n大于1时，`isPowerOfThree(n) = n % 3 === 0 && isPowerOfThree(n / 3)`

### code:
```javascript
/**
 * @param {number} n
 * @return {boolean}
 */
var isPowerOfThree = function(n) {
    if (n < 1) {
        return false
    } else if (n === 1) {
        return true
    } else {
        return n % 3 === 0 && isPowerOfThree(n / 3)
    }
}
```
