# [有效的括号](https://leetcode-cn.com/leetbook/read/top-interview-questions-easy/xnbcaj/)

### algorithm:
- 使用一个栈并遍历字符串
- 遇到左括号入栈，遇到右括号出栈并判断是否对应
- 遍历结束后如栈深度为0则有效

### code:
```javascript
/**
 * @param {string} s
 * @return {boolean}
 */
var isValid = function(s) {
    const stack = []
    for (const c of s) {
        if (['(', '[', '{'].includes(c)) {
            stack.push(c)
        } else {
            const left = stack.pop()
            if (c === ')' && left !== '(') {
                return false
            } else if (c === ']' && left !== '[') {
                return false
            } else if (c === '}' && left !== '{') {
                return false
            }
        }
    }
    return stack.length === 0
}
```
