# [汉明距离]()

### algorithm:
- `x^y`可得一个二进制数，每一位为0表示x和y在该位上相同，为1则不同
- 对该数取汉明重量即为x和y的汉明距离

### code:
```javascript
/**
 * @param {number} x
 * @param {number} y
 * @return {number}
 */
var hammingDistance = function(x, y) {
    return hammingWeight(x^y)
};
var hammingWeight = function(n) {
    let count = 0
    while (n > 0) {
        count += (n % 2)
        n >>>= 1
    }
    return count
}
```
