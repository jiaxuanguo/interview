# [位1的个数](https://leetcode-cn.com/leetbook/read/top-interview-questions-easy/xn1m0i/)

### algorithm:
- 如果当前的数能被2整除，则其末位为0，否则为1
- 当前数除2并取整即可去掉末位，当然也可以用`>>>`
- 循环至当前数为0

### code:
```javascript
/**
 * @param {number} n - a positive integer
 * @return {number}
 */
var hammingWeight = function(n) {
    let count = 0
    while (n > 0) {
        count += (n % 2)
        n >>>= 1
    }
    return count
}
```
