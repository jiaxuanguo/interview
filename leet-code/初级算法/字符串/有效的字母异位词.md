# [有效的字母异位词](https://leetcode-cn.com/leetbook/read/top-interview-questions-easy/xn96us/)

### algorithm:
1. 遍历字符串s，利用HashMap记录每个字符出现的次数
2. 遍历字符串t，若某个字符在map中的value为0则返回false，否则将其减一
3. 遍历map的value，若都为0则返回true，否则返回false

### code:
```javascript
/**
 * @param {string} s
 * @param {string} t
 * @return {boolean}
 */
var isAnagram = function(s, t) {
    const map = new Map()
    for (const c of s) {
        const count = map.get(c) ?? 0
        map.set(c, count + 1)
    }
    for (const c of t) {
        const count = map.get(c) ?? 0
        if (count === 0) {
            return false
        }
        map.set(c, count - 1)
    }
    return Array.from(map.values()).every(i => i === 0)
};
```
